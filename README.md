# Soulhouse × Default Resilient Method — Pilot Landing

Two language versions of the pilot landing page, deployable on GitHub Pages.

## Files

- `index.html` — English (default)
- `de.html` — German
- `sh.css`, `sh-additions.css` — page styles
- `assets/` — brand tokens, hero video, portrait

## Deploy to GitHub Pages

1. Create a new GitHub repo (e.g. `soulhouse-pilot`)
2. Upload everything in this `site/` folder to the repo root
3. In the repo: **Settings → Pages → Branch: main / (root) → Save**
4. Wait ~1 min. Your pages go live at:
   - `https://<your-username>.github.io/soulhouse-pilot/` (EN)
   - `https://<your-username>.github.io/soulhouse-pilot/de.html` (DE)

Language switcher in the top-right lets viewers toggle between EN and DE.
